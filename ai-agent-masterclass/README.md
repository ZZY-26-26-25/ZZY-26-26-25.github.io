# AI 智能体大师课

面向零基础学习者的免费交互式课程。课程以“旅行规划智能体”为贯穿项目，从 AI 与 LLM 基础逐步推进到工具调用、资料检索、RAG、旅行偏好记忆、完整行程与 Plan B 规划、多智能体协作、评测、安全审批和部署。社交平台只用于发现摄影机位与体验线索；开放状态、安全和交通等关键事实必须由官方或一手来源交叉验证。

## 当前版本

- 版本：`v0.2.0`
- 更新时间：`2026-07-22`
- 已发布：导学课、第 1 课
- 站点形态：无依赖静态站，可直接由 GitHub Pages 托管

## 文件

- `index.html`：网站外壳与无障碍结构
- `styles.css`：响应式视觉系统
- `course-data.js`：课程、测验、术语和更新日志
- `app.js`：路由、搜索、进度、小测及互动实验

学习进度和 Agent Charter 默认只存于访问者的 `localStorage`；网站不上传这些数据。需要真实模型 API 的后续实验不会在 GitHub Pages 前端保存密钥。

## 每课发布协议

每次教学完成后：

1. 新增或修订课程正文。
2. 同步互动实验、小测、术语和一手来源。
3. 更新版本号、验证日期和更新日志。
4. 检查脚本语法、内容数据、链接与核心交互。
5. 提交 GitHub，等待 Pages 发布并验证公开页面。

## 本地预览

在本目录启动任意静态文件服务器即可，例如：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173/`。
